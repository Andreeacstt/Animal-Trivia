About the website


This is a quiz website, consisting of 10 questions, all of them animal related.
On the home page, there is a navigation bar on the right upper corner, that will
take you to Author Page, Contact Page, and About Page, as well as the button
that opens the quiz. On the first page, you will be asked for a name in order for
the quiz to start. The questions are multiple-choice, with four possible answers,
where only one has to be selected. Throughout the quiz there are buttons for
“previous” and “next” questions. When all the questions are answered, the
submit button will prompt the results. There is also a "play again" button that
will restart the quiz.
